## Purpose
This repository contains **store-specific CSS files** used to quickly fix layout or styling issues on individual stores.  
These files are intended for **temporary solutions only**, to address urgent problems without altering the main theme.

---

## Workflow

1. **Permission Required:**  
   Before any changes, the store owner must grant us permission to modify their theme.

2. **File Creation:**  
   Once permission is granted, a CSS file is created for the store, named using the store ID
